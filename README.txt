Для успішного запуску сайту потрібно:
1. Запустити локальний сервер. Для цього потрібно запустити "run.bat" або "run.vbs"
2. Зайти на сам сайт. Запустивши яклик "website" або просто перейти за посиланням http://localhost:3000/
3. Після закінчення роботи треба зупинити локальний сервер. Для цього потрібно запустити "stop.bat" або "stop.vbs"
